1. Inline CSS

Inline CSS is applied directly to a specific HTML element using the style attribute.Usage: Best for quick, one-off style changes or testing specific elements.
Where and How to use: It's used directly within the opening tag of element using style attribute

Where and How to use: We need to create separate file and link it with html using tag / element
